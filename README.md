# ng-file-upload-client
Angularを使ったファイルアップロードの1つの例.

# Usage
```bash
git clone git@github.com:solareenlo/ng-file-upload-client.git
cd ng-file-upload-client
docker-compose up -d
```
`localhost:4204`で走ってる.
